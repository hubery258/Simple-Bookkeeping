# Simple-Bookkeeping 简单记账

***

一个轻量、无广告、可在手机端使用的记账应用，支持快速记录收支，计算特定时间段的总收支情况

## 项目目标 
- 简洁高效地记录每一笔支出 
- 支持手机端安装与离线使用（PWA）

## 项目进度 

### MVP
- [ ] 添加支出/收入
- [ ] 查看账目列表
- [ ] 分类系统与每单备注
- [ ] 简单统计
- [ ] 移动端适配
- [ ] PWA 支持

### Future Features
- [ ] 账户管理
- [ ] 预算提醒
- [ ] 数据导入/导出
